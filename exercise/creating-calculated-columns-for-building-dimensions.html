Creating Calculated Columns for Building Dimensions

üßÆ Function: Arithmetic Operators (+, /), DIVIDE

Difficulty: Beginner

üìö Theory & Explanation

Calculated columns are new columns added to a table in your data model, with values computed row-by-row based on a DAX formula. They are useful for persisting row-level calculations that you want to use in various visuals, filters, or as inputs for other measures. The DIVIDE function is a best practice for division in DAX as it gracefully handles division by zero errors by returning BLANK, preventing common errors that the standard division operator (/) might produce.

üß© Example Use Case

As Power BI analysts, you'll constantly perform calculations; the basic arithmetic `+` operator allows you to sum values, like combining 'Product Sales' with 'Service Sales.' While the `/` operator works for straightforward division (e.g., 'Total Sales / Number of Customers'), the `DIVIDE` function is your best practice for any division calculation, such as 'Profit Margin' or 'Conversion Rate.' `DIVIDE` robustly handles potential division-by-zero scenarios, preventing errors in your reports by returning `BLANK()` or a specified alternate result, ensuring clean and reliable visuals.

üéØ Your Task

1. Download and unzip the 'Tallest Buildings - Calculated Columns.pbix' file. 2. Open the `.pbix` file in Power BI Desktop and switch to the Table view for the 'Building' table. 3. From the 'Table tools' ribbon, select 'New column'. 4. Rename the new column to 'Total floors' and enter the DAX formula: `='Building'[Floors above ground] + 'Building'[Floors below ground]`. Press Enter. 5. Create another new calculated column, renaming it 'Average floor height'. 6. Enter the DAX formula: `=DIVIDE('Building'[Height m], 'Building'[Floors above ground])`. Press Enter. 7. With the 'Average floor height' column selected, go to the 'Column tools' ribbon and format it to display with two decimal places. 8. Switch to the Report view and add a table visual. 9. Add relevant columns to the table visual, including 'Building Name', 'Height m', 'Total floors', and 'Average floor height'. Sort the table as desired. 10. Save your Power BI Desktop file.

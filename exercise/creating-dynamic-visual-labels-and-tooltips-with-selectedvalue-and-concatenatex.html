Creating Dynamic Visual Labels and Tooltips with SELECTEDVALUE and CONCATENATEX

üßÆ Function: SELECTEDVALUE, CONCATENATEX

Difficulty: Beginner

üìö Theory & Explanation

The SELECTEDVALUE function returns the value when there's only one distinct value in the specified column, otherwise it returns an alternate result (or BLANK). This is ideal for dynamic titles. The CONCATENATEX function iterates over a table, evaluates an expression for each row, and concatenates the results into a single string, allowing for custom separators and sorting. This is highly useful for detailed tooltips or lists within a visual.

üß© Example Use Case

To build truly dynamic and informative report titles or text cards, `SELECTEDVALUE` and `CONCATENATEX` are often combined. Imagine wanting a visual title that displays the *specific product category* when only one is filtered, but then intelligently lists *all selected categories* if the user picks multiple, or a generic 'All Categories' if none are chosen. This powerful pairing allows your visuals to adapt, providing precise context to your audience based on their filter selections.
